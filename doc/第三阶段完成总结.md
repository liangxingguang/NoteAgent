## 🎯 核心功能

### 1. 双模式支持
现在飞书适配器支持两种模式：

#### Webhook 模式（推荐）
```env
FEISHU_USE_WEBHOOK=true
FEISHU_WEBHOOK_HOST=0.0.0.0
FEISHU_WEBHOOK_PORT=8000
```

**特点：**
- ⚡ 实时消息推送
- 📉 低资源占用
- 🔒 需要公网 IP
- ✅ 飞书官方推荐

#### 轮询模式（兼容性）
```env
FEISHU_USE_WEBHOOK=false
FEISHU_POLL_INTERVAL=1.0
```

**特点：**
- ✅ 无需公网 IP
- 🐌 有延迟
- 📈 较多 API 调用

### 2. 完整的 Webhook 服务器
```python
from platforms import FeishuWebhookServer

server = FeishuWebhookServer(
    verification_token="your_token",
    message_callback=your_callback,
)

await server.start(host="0.0.0.0", port=8000)
```

**端点：**
- `POST /feishu/webhook` - Webhook 接收端点
- `GET /health` - 健康检查

### 3. 请求验证

## 🚀 使用示例

### 快速配置 Webhook 模式

1. 编辑 `.env`：
```env
FEISHU_ENABLED=true
FEISHU_APP_ID=cli_xxxxxxxxxxxxx
FEISHU_APP_SECRET=xxxxxxxxxxxxx
FEISHU_VERIFICATION_TOKEN=xxxxxxxxxxxxx
FEISHU_ALLOWED_USER_IDS=ou_xxxxxxxxxxxxx

FEISHU_USE_WEBHOOK=true
FEISHU_WEBHOOK_HOST=0.0.0.0
FEISHU_WEBHOOK_PORT=8000
```

2. 启动程序：
```bash
python main_multi.py
```

3. 在飞书开放平台配置 Webhook URL：
```
https://your-domain.com/feishu/webhook
```

4. 开始使用！
