## 🎯 核心功能

### 1. 富文本消息支持
- **RichTextBuilder**: 链式调用构建富文本
- 支持样式：粗体、斜体
- 支持元素：文本、链接、@用户、图片
- 内置常用消息模板

### 2. 命令处理系统
- **命令注册表**: 可扩展的命令注册机制
- **内置命令**:
  - `/start` / `/hello`: 显示欢迎消息
  - `/help` / `?`: 显示帮助信息
  - `/ping`: 测试服务在线
  - `/about`: 显示机器人信息
- **命令别名**: 支持多个名称映射到同一命令

### 3. 群聊支持
- 自动识别群聊/单聊消息
- 正确的消息发送类型处理
- @机器人触发

### 4. 增强的消息发送
- 快捷方法，简化调用
- 自动处理富文本/纯文本选择
- 更好的用户体验

## 📖 使用示例

### 使用富文本构建器
```python
from platforms.feishu_rich_text import RichTextBuilder

builder = RichTextBuilder(title="欢迎")
builder.add_text("Hello, ", bold=True).add_text("World!")
builder.new_line()
builder.add_link("点击访问", "https://example.com")

rich_text_content = builder.build()
await adapter.send_message(chat_id, rich_text_content, use_rich_text=True)
```

### 使用快捷方法
```python
# 发送欢迎消息
await adapter.send_welcome_message(chat_id)

# 发送帮助消息
await adapter.send_help_message(chat_id)

# 发送成功消息
await adapter.send_success_message(chat_id, "note_2024.md", "我的笔记")

# 发送错误消息
await adapter.send_error_message(chat_id, "文件太大了")
```

### 注册自定义命令
```python
from platforms.feishu_commands import command_registry

async def my_command_handler(handler, args, ctx):
    return f"你调用了我的命令！参数: {args}"

# 注册命令
command_registry.register(
    name="mycmd",
    handler=my_command_handler,
    help_text="我的自定义命令",
    aliases=["mc", "m"]  # 别名
)
```

## 🚀 快速开始（飞书）

### 配置
```env
# 启用飞书
FEISHU_ENABLED=true

# 飞书应用凭证
FEISHU_APP_ID=cli_xxxxxxxxxxxxx
FEISHU_APP_SECRET=xxxxxxxxxxxxx
FEISHU_VERIFICATION_TOKEN=xxxxxxxxxxxxx

# Webhook 模式（推荐）
FEISHU_USE_WEBHOOK=true
FEISHU_WEBHOOK_HOST=0.0.0.0
FEISHU_WEBHOOK_PORT=8000

# 授权用户
FEISHU_ALLOWED_USER_IDS=ou_xxxxxxxxxxxxx
```

### 启动
```bash
# Windows
.venv\Scripts\python.exe main_multi.py

# Linux/Mac
source .venv/bin/activate
python main_multi.py
```

### 在飞书中使用
1. 给机器人发送 `/start` 查看欢迎
2. 直接发送文本、链接或文件
3. 输入 `/help` 获取帮助

## 🔧 可扩展功能

### 如何添加自定义命令
```python
from platforms.feishu_commands import command_registry

async def handle_my_command(handler, args, ctx):
    # ctx: {user_id, chat_id, message_id, platform}
    user_name = ctx.user_name or "用户"
    return f"你好, {user_name}！参数: {args}"

# 注册命令
command_registry.register(
    name="mycmd",
    handler=handle_my_command,
    help_text="我的命令",
    aliases=["mc"]
)
```

### 如何扩展富文本
```python
from platforms.feishu_rich_text import RichTextBuilder

builder = RichTextBuilder(title="自定义消息")
builder.add_text("这里是粗体", bold=True).new_line()
builder.add_text("这里是斜体", italic=True).new_line()
builder.add_link("点击这里", "https://example.com")

# 发送
content = builder.build()
await adapter.send_message(chat_id, content, use_rich_text=True)
```
